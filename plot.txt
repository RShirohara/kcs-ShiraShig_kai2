title: 未定
題材: ボトルメール、手紙を通じた出会い
- 瓶に入った手紙を流すと、流れ流れてどこかの相手へ届くらしい
- なぜかいつも同じ相手へと届く -> 誰かが仲介している等(冗談交じりに)噂が広がる
  (潜水艦娘、もしかしたら深海棲艦)
- 静かな噂として、一部の艦娘のみに伝わる
- 白露: どこかの泊地(南方)
- 時雨: どこかの鎮守府(国内)

はじまり
1. 白露
  噂を聞く
  姉妹艦が来ていない寂しさ、暇つぶしも兼ねる
  同じ相手だと気づかれるように印をつける
  中身は静かめに、一見して白露と気が付かない
2. 時雨
  砂浜を歩いていると、瓶を見つけて拾い上げる
  印あり(白露のもの)
  噂はここまで流れている -> 内容が変化している(噂は中身が変わるもの)
  あまり信じていない
  読んでみると、自分と近しい艦娘が書いた文章だとわかる -> 直感、艦娘の勘?
  返事を出す
  相手と自分の印とを両方つける -> 仲介する何かにわかるように

転換
3. 白露
  夜の砂浜
  月の光で照らされる瓶
  人の目を盗んで抜け出してきた白露
  その場で手紙を読む -> 相手がだれかを読み解こうとする
  書かれた文章の手触りから、相手の暮らす街の情景を読み取ろうとする
  シーン転換: 後日、部屋
  手紙の内容を考えている
  まだ名前は書いていない(関係だけ)
    それでも伝わる。
    相手のことについて考える
    相手は自分に気付いているのかと考える
    気付けるのは時雨だけ
  敵襲
    書きかけの手紙を出そうか迷う
    -> 「急ぎの用事が出来てしまったので、今回はこれくらいで」等走り書きして瓶に入れる -> 持ったまま出撃
  or
    すでに砂浜にいる、瓶を流そうとする直前
    襲撃に遭いケガ、瓶割れる

4. 時雨
  ある時を境にぱったりと手紙が途切れる
  待ち続ける
  これまでの手紙を読み返す -> 文章から相手の住む場所を想像する
  回想: 瓶を開けると南方の空気(砂、貝殻?) 
  南方で深海棲艦との大規模な戦闘があった(情報、ニュース)
  砂浜で待つ
  目の前の海は、自分の知らない海へと通じているのかと思う

5. 締め
  白露
    病院で手紙を書く
    しばらく出せなかったことの謝罪、戦闘で負傷して帰国すること等
    ひょっとしたら会えるかもしれないと書きつつ、流石に無いと思う
    手紙の中身を一部
    残骸の残る砂浜から瓶を流す
    久しぶりに故郷の海に行こうかと思う(時雨のいる砂浜)
  時雨
    場所: 砂浜
    日課の散歩
    数日前に届いた手紙について考えている(戦闘前に白露が出したもの)
    無事だろうかと考える
    何かが水面下で動く -> 目の前で瓶が流れ着くのを見る
    印に気が付く -> 読む
    手紙の中身を一部、上と同じ個所
    背後の誰かに気が付き振り向く -> 白露
      白露: 手元の瓶に気が付く -> 相手だと分かる
      時雨: 反応で気が付く
    お互いに姉妹艦だと気が付く
    向かい合って挨拶を交わす -> 姉妹の初めての出会い
